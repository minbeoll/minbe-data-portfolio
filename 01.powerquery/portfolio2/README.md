📊 Excel Automation System (Power Query)

Overview

This project demonstrates an automated Excel reporting system built entirely with Power Query.  It consolidates multiple monthly Excel files, applies business logic, performs automated mapping, detects unmapped records, and generates reports — all with a single refresh. Designed for real-world business environments where data is updated regularly and manual processing is inefficient.



Key Features 



Automatic folder-based file consolidation
Dynamic mapping using merge logic
Unmapped data detection (data governance)
Auto-updating reports (monthly, category, Top N)
Error-tolerant refresh structure



Workflow 



Add new monthly Excel file
Click Refresh
Mapping + validation applied automatically
Reports update instantly
No manual processing required.



Tech Stack



Microsoft Excel
Power Query (M language)
Folder-based ETL
Merge logic + Anti Join detection



Business Value



Eliminates repetitive manual work
Reduces human error
Ensures consistent reporting
Scales with growing data



📊 엑셀 자동화 시스템 (Power Query)



개요

이 프로젝트는 Power Query만으로 구축한 엑셀 자동 보고 시스템입니다. 여러 개의 월별 Excel 파일을 자동 통합하고, 매핑 로직을 적용하며, 매핑되지 않은 데이터를 탐지하고, 보고서를 자동 생성합니다. 모든 작업은 새로고침 한 번으로 완료됩니다. 실제 업무 환경에서 반복적인 수작업을 제거하기 위해 설계되었습니다. 



주요 기능



폴더 기반 파일 자동 통합
Merge 기반 자동 매핑 로직
미매핑 데이터 자동 탐지
월별 / 카테고리 보고서 자동 생성
오류 대응 가능한 구조



사용 방법



월별 파일 폴더에 추가
새로고침 클릭
매핑 및 검증 자동 수행
보고서 자동 업데이트
수작업 데이터 처리 불필요



기술 스택



Microsoft Excel
Power Query (M 언어)
폴더 기반 ETL
Merge + Anti Join 로직



비즈니스 가치



반복 업무 제거
인적 오류 감소
보고 일관성 향상
데이터 증가에도 확장 가능
