# 🧩 Templates Library

This folder contains reusable templates and building blocks used across my projects.

These templates help accelerate development and ensure consistent architecture across automation and analytics workflows.

---

## 🚀 Purpose

Instead of rebuilding everything from scratch, I maintain a collection of reusable templates for:

- Faster project delivery
- Consistent structure
- Scalable automation systems
- Maintainable codebases

This approach enables rapid prototyping while keeping professional standards.

---

## 🧠 Template Philosophy

All templates follow a few core principles:

- Modular design
- Plug-and-play usability
- Clear folder architecture
- Real-world usability (not toy examples)

They are built from actual project experience and refined over time.

# 🇰🇷 템플릿 라이브러리

이 폴더는 프로젝트 전반에서 사용하는 재사용 가능한 템플릿 모음입니다.

템플릿을 통해 빠른 개발과 일관된 구조를 유지하며  
자동화 및 데이터 프로젝트의 생산성을 높이고 있습니다.

---

## 📌 목적

매번 처음부터 만드는 대신, 재사용 가능한 템플릿을 구축하여:

- 개발 속도 향상
- 구조 일관성 유지
- 확장 가능한 자동화 설계
- 유지보수 가능한 코드

를 목표로 합니다.

---

## 🧠 템플릿 철학

모든 템플릿은 다음 기준을 따릅니다:

- 모듈형 설계
- 쉽게 재사용 가능
- 명확한 폴더 구조
- 실전 기반 템플릿

단순 예제가 아니라 실제 프로젝트에서 검증된 구조입니다.

---

## 📦 템플릿 종류

### 📊 데이터 & BI
- Power BI 기본 구조
- DAX 라이브러리
- Power Query 템플릿

### 🐍 자동화
- Python 자동화 기본 틀
- 파일 처리 템플릿
- Excel 자동화 스타터

### 🌐 데이터 엔지니어링
- 웹 크롤링 템플릿
- API 수집 구조
- ETL 파이프라인 스타터

### 🖥️ 앱 & 도구
- Streamlit 앱 템플릿
- AI 어시스턴트 구조
- 프로젝트 생성 템플릿

## 🛠 템플릿의 가치

템플릿을 통해:
