# 📂 Portfolio Projects

This folder contains real-world data automation and dashboard projects.

Each project represents an **End-to-End workflow**, from raw data to final insights.

---

## 🚀 What You’ll Find Here

Projects in this folder typically include:

- Power BI dashboards
- Power Query automation
- Python automation
- Excel automation
- Web crawling pipelines
- API-based data ingestion

Each project focuses on solving a **practical business problem**, not just technical demonstrations.

---

## 🧠 Project Philosophy

Every project follows the same core approach:

**Raw Data → Cleaning → Automation → Visualization**

I prioritize:
- Automation over manual workflows
- Reproducible pipelines
- Clean structure and maintainability

These are built with real client scenarios in mind.

---

## 📁 Typical Project Structure

Each project may include:

project-name/
│
├── data/ # Sample or processed data (if shareable)
├── dax/ # DAX measures (if Power BI project)
├── mcode/ # Power Query scripts
├── python/ # Automation or crawling scripts
├── snapshot/ # Dashboard preview images
├── PowerBI.pbix # Main dashboard file (via Git LFS)
└── README.md # Project-specific explanation


Structure may vary depending on the project type.

---

## ⚠️ Notes

- Some raw datasets may be excluded due to size or privacy
- PBIX files are tracked using Git LFS
- Sample data is included when possible

# 🇰🇷 프로젝트 안내

이 폴더에는 실제 데이터 자동화 및 대시보드 포트폴리오가 포함되어 있습니다.

각 프로젝트는 단순한 결과물이 아니라  
**데이터 수집부터 자동화, 시각화까지 전체 흐름(End-to-End)** 을 보여줍니다.

---

## 📌 포함 내용

프로젝트에는 다음이 포함될 수 있습니다:

- Power BI 대시보드
- Power Query 자동화
- Python 자동화 코드
- Excel 자동화
- 웹 크롤링 파이프라인
- API 기반 데이터 수집

실제 문제 해결 중심 프로젝트로 구성되어 있습니다.

---

## 🧠 설계 철학

모든 프로젝트는 동일한 흐름을 따릅니다:

데이터 → 정제 → 자동화 → 시각화

중요하게 생각하는 요소:
- 수작업 최소화 (자동화 중심)
- 재현 가능한 구조
- 유지보수 가능한 설계

---

## 📁 기본 구조

프로젝트는 일반적으로 다음과 같은 구조를 가집니다:

- data → 샘플 데이터
- dax → DAX 수식
- mcode → Power Query 코드
- python → 자동화 코드
- snapshot → 대시보드 미리보기
- pbix → Power BI 파일
- README → 프로젝트 설명

프로젝트 유형에 따라 일부 구조는 달라질 수 있습니다.
