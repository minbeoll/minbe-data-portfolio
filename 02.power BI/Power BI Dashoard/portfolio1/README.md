Description – HR Analytics Dashboard (3 Pages)

   A comprehensive 3-page Power BI HR Analytics dashboard providing insights into workforce demographics, hiring trends, and retention performance.
   The report visualizes gender distribution, age group composition, employment types, monthly new hires and leavers, departmental turnover activity, and cumulative headcount trends.
   Interactive slicers allow users to filter by month and year, enabling deeper exploration of seasonal hiring patterns and workforce changes.
   The dashboard highlights key HR KPIs—such as active headcount, net change, retention rate, and turnover rate—using clean visual layouts optimized for readability and executive decision-making.
