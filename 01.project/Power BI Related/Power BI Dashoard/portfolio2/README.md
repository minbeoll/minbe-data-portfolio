Description

   A 3-page interactive Power BI report covering sales performance, global market distribution, profit analysis, and return trends.
   It includes KPI cards, detailed category insights, a world map with regional filters, weekly profit comparison with price adjustment simulation, and product-level return metrics.
 Designed with a clean, modern layout to maximize readability and deliver clear business storytelling.



&nbsp;  매출 성과, 글로벌 시장 분포, 수익 분석, 반품 추이를 종합적으로 다룬 3페이지 구성의 인터랙티브 Power BI 보고서입니다. 본 보고서는 KPI 카드, 카테고리별 상세 인사이트, 지역 필터가 적용된 세계 지도, 가격 조정 시뮬레이션을 포함한 주간 수익 비교, 제품 단위 반품 지표 등을 포함하여 비즈니스 전반을 다각도로 분석할 수 있도록 구성되었습니다. 또한 가독성을 극대화한 깔끔하고 현대적인 레이아웃으로 설계하여 명확한 비즈니스 스토리텔링을 효과적으로 전달하도록 제작되었습니다.
