# Power BI refresh-update



Portfolio1



demo-video https://youtu.be/NBdDz8mlAoY



   I created a 2023.csv file using Python to verify that the refresh and update process works correctly in my Power BI portfolio. After placing the 2023.csv file into the sales data folder, I performed a refresh in my Power BI portfolio.

As a result, the data from 2023.csv appeared correctly and the charts in my Power BI report were updated as expected. This confirms that the refresh mechanism is working properly. Regardless of which data file is loaded into the Power BI (.pbix) file, as long as the data is refreshed and updated, I can ensure that the report works correctly.



&nbsp;  Power BI 포트폴리오에서 새로 고침 및 업데이트 프로세스가 정상적으로 작동하는지 검증하기 위해 Python을 사용하여 2023.csv 파일을 생성하였습니다. 이후 해당 2023.csv 파일을 sales data 폴더에 추가한 뒤 Power BI 포트폴리오에서 새로 고침을 수행하였습니다. 그 결과, 2023.csv의 데이터가 정상적으로 반영되었고 Power BI 보고서의 차트 또한 예상대로 업데이트되었습니다. 이를 통해 새로 고침 메커니즘이 올바르게 작동함을 확인할 수 있었습니다. 즉, 어떤 데이터 파일이 Power BI(.pbix)에 로드되더라도 새로 고침과 업데이트가 정상적으로 이루어진다면 보고서가 문제없이 동작함을 보장할 수 있습니다.
