# Minbe Data Portfolio

End-to-end data automation projects built with Python, MySQL, Power Query, and Power BI.

This repository contains real-world portfolio projects covering:
- Web crawling & API data ingestion
- Database modeling (MySQL)
- Power Query automation
- Power BI dashboards
- ETL monitoring & reporting automation

All projects are designed for freelance and real business scenarios, focusing on automation, scalability, and clean architecture.

# 민별의 데이터 포트폴리오

Python, MySQL, Power Query, Power BI를 활용한 엔드투엔드 데이터 자동화 포트폴리오입니다.

이 저장소에는 다음 내용을 포함합니다:
- 웹 크롤링 및 API 데이터 수집
- MySQL 데이터 모델링
- Power Query 자동화
- Power BI 대시보드
- ETL 모니터링 및 리포팅 자동화

실제 프리랜서 및 실무 환경을 기준으로 자동화, 확장성, 구조 설계를 중심으로 제작되었습니다.
